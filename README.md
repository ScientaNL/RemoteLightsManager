RemoteLightsManager
===================

Remote Lights Manager klik aan klik uit webinterface voor het verkeerslicht.
Met deze code wordt het verkeerslicht in het kantoor van Syslogic aangestuurd, via een webinterface.
Dit project is sequencer van shell commando's die Klik aan klik uit lampen aansturen. De code wordt ontsloten via een webinterface.
